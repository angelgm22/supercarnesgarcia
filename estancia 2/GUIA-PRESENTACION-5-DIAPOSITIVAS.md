# GUÍA PARA PRESENTACIÓN - 5 DIAPOSITIVAS
## Super Carnes García - Sitio Web Promocional

---

## DIAPOSITIVA 1: PÁGINA PRINCIPAL

**Screenshot a tomar:**
- Captura de pantalla completa de la página de inicio del sitio web
- Asegúrate de que se vea el menú de navegación (Inicio, Productos, Carnes, Ofertas)
- Incluye el contenido principal de bienvenida

**Texto de la diapositiva:**

**Título:** Sitio Web Promocional - Página Principal

**Contenido:**
Se desarrolló un sitio web moderno para Super Carnes García donde los clientes pueden conocer todos los productos disponibles. La interfaz es limpia, profesional y fácil de navegar, con acceso directo a las diferentes secciones del catálogo mediante el menú principal.

---

## DIAPOSITIVA 2: CATÁLOGO DE PRODUCTOS

**Screenshot a tomar:**
- Captura de la página de Productos o Carnes mostrando las tarjetas de productos
- Muestra varios productos con sus imágenes, nombres y precios
- Asegúrate de que se vean bien las tarjetas organizadas

**Texto de la diapositiva:**

**Título:** Catálogo de Productos

**Contenido:**
El catálogo presenta todos los productos de manera visual y organizada. Cada producto muestra su imagen, nombre y precio claramente. Los clientes pueden explorar fácilmente toda la oferta disponible, facilitando la planificación de sus compras antes de visitar la tienda física.

---

## DIAPOSITIVA 3: PRODUCTOS EN OFERTA

**Screenshot a tomar:**
- Captura de la página de Ofertas
- Muestra productos con el precio normal tachado y el precio de oferta destacado
- Si es posible, incluye varios productos en oferta para mostrar la variedad

**Texto de la diapositiva:**

**Título:** Sistema de Ofertas y Promociones

**Contenido:**
Se implementó un sistema de precios con ofertas que permite destacar productos en promoción. Los clientes pueden identificar fácilmente los descuentos con el precio normal tachado y el precio de oferta en color destacado. Esta sección especial atrae la atención hacia las mejores oportunidades de compra.

---

## DIAPOSITIVA 4: BÚSQUEDA PREDICTIVA

**Screenshot a tomar:**
- Captura de la barra de búsqueda en acción
- Muestra la búsqueda con texto escrito y resultados apareciendo en tiempo real
- Puede ser un GIF o captura mostrando sugerencias predictivas

**Texto de la diapositiva:**

**Título:** Búsqueda Inteligente de Productos

**Contenido:**
El sistema cuenta con búsqueda predictiva en tiempo real que permite a los clientes encontrar productos específicos de manera rápida y eficiente. Los resultados se muestran instantáneamente mientras el usuario escribe, mejorando significativamente la experiencia de navegación y ahorrando tiempo.

---

## DIAPOSITIVA 5: DISEÑO RESPONSIVO

**Screenshot a tomar:**
- Captura mostrando el sitio en diferentes dispositivos (desktop, tablet, móvil)
- Puede ser una imagen dividida mostrando las tres vistas
- O tres capturas separadas del mismo contenido en diferentes tamaños de pantalla

**Texto de la diapositiva:**

**Título:** Accesible desde Cualquier Dispositivo

**Contenido:**
El sitio web cuenta con diseño responsivo que se adapta automáticamente a cualquier dispositivo. Ya sea desde computadora, tablet o smartphone, los clientes disfrutan de una experiencia óptima con navegación fluida y contenido perfectamente ajustado a cada pantalla. Esto garantiza accesibilidad 24/7 desde cualquier lugar.

---

## TIPS PARA LAS CAPTURAS:

1. **Usa un navegador limpio** sin extensiones visibles en la barra
2. **Maximiza la ventana** para mostrar el sitio completo
3. **Asegúrate de que haya productos** con imágenes visibles
4. **Captura en buena resolución** (mínimo 1920x1080)
5. **Para la vista responsiva:** Usa las herramientas de desarrollador (F12) para simular diferentes dispositivos

## ORDEN DE PRESENTACIÓN:

1. Presenta la página principal (primera impresión)
2. Muestra el catálogo (funcionalidad principal)
3. Destaca las ofertas (valor agregado)
4. Demuestra la búsqueda (facilidad de uso)
5. Cierra con responsividad (accesibilidad total)
